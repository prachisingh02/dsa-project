Data Structures and Algorithms (DSA) in Python

Overview
This repository contains Python programs for basic Data Structures and Algorithms (DSA). All programs were written and executed using Visual Studio Code (VS Code) as part of my learning and academic practice.

1. Arrays
- Array creation
- Traversal
- Insertion
- Deletion
- Searching

 2. Linked Lists
- Singly Linked List
- Insertion Operations
- Deletion Operations
- Traversal

3. Stacks
- Stack implementation using Python lists
- Push operation
- Pop operation
- Peek operation

4. Queues
- Queue implementation
- Enqueue operation
- Dequeue operation

 5. Searching Algorithms
- Linear Search
- Binary Search

 6. Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort


Technologies Used
- Python 3.x
- Visual Studio Code (VS Code)
